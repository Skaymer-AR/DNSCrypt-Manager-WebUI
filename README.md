# DNSCrypt Manager

**Creado por Skaymer AR**

Módulo root para Android con `dnscrypt-proxy`, WebUI para KernelSU/KernelSU Next/APatch, CLI para Magisk y recuperación segura de red.

⏳ El repositorio se está inicializando automáticamente. La GitHub Action incluida extraerá el código fuente, verificará el binario oficial ARM64, ejecutará las pruebas y publicará la primera release.
