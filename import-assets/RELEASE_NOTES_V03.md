# DNSCrypt Manager v0.3.0-RC1-WIP-A — vista previa

> **Vista previa de desarrollo. No es estable ni está completa.** La primera versión estable será v1.0.0.

## Advertencias obligatorias

- **BindHosts debe permanecer desactivado** mientras DNSCrypt Manager esté activo. Desactivalo, reiniciá y recién entonces instalá o habilitá este módulo. La combinación puede causar pérdida de conectividad o bootloop.
- En KernelSU Next puede ser necesario activar **Hybrid Mount** y reiniciar.
- No usar como actualización de producción ni como reemplazo de una versión estable.

## Incluye hasta Checkpoint A

- Nueva WebUI tipo SPA.
- Navegación inferior fija: Estado, DNS, Listas, Actividad y Ajustes.
- Rutas hash y soporte del botón Atrás.
- Inglés predeterminado y español seleccionable.
- Preferencia de idioma persistente y fallback a inglés.
- Arquitectura y plan de trabajo v0.3.
- Correcciones de catálogo y entorno heredadas de RC2.2.

## Todavía no incluye

- Anonymized DNSCrypt administrable.
- ODoH administrable y probado en Android.
- Portal cautivo.
- Monitor de actividad sospechosa.
- Políticas DNS por aplicación.
- `source doctor` y descarga bootstrap completos.

Esta publicación existe para reflejar el progreso real del repositorio y facilitar pruebas controladas.
