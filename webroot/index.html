"Redirigir" usa el mismo proxy para IPv4 e IPv6. "Bloquear" corta las consultas DNS por IPv6 para que no se escapen sin cifrar; el resto del trafico IPv6 no se toca.

Proteccion web

Bloqueo de dominios por categoria. Malware, phishing y estafas vienen activados; rastreadores, publicidad y criptomineria, desactivados para no romper apps ni paginas legitimas.

Listas de bloqueo

Toca “Actualizar activas” para descargar y verificar las listas.

Cada actualizacion se descarga a un archivo temporal, se verifica (tamaño, SHA-256, sintaxis), se prueba con dnscrypt-proxy y solo entonces se aplica. Si algo falla, se restaura la version anterior automaticamente.

Privacidad e historial

Todo el historial es local: nada sale del telefono, no hay telemetria. DNS cifrado no es una VPN, DNSSEC no vuelve segura una pagina, y bloquear dominios no reemplaza a un antivirus. Detalle en PRIVACY.md.
